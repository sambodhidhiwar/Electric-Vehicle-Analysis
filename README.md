## 📊 Overview
This project analyzes Electric Vehicle (EV) data to uncover trends in adoption, technology, and market share.

## 🎯 Key Insights
- Total Vehicles: 150K+
- BEVs dominate with ~77%
- Increasing trend from 2010–2023
- Tesla leads the EV market

## 📈 Dashboard Features
- Vehicles by Model Year (Growth Trend)
- Vehicles by State (Geographical Analysis)
- Top 10 Manufacturers
- CAFV Eligibility Distribution
- Top Vehicle Models
